1. What is HTTP?

HTTPS is short for "Hypertext Transfer Protocal". it means that the connection would be encrypted between your web browswer and the website. Though this does not always mean a website is secure.

2. What is HTML?

HTML (Hypertext Markup Language) is a coding language that is used to build websites across the internet.

3. Describe the process of requesting a web page from a web server.

Your browser goes to the DNS server, gets the IP address of the website, and then sends back a copy of that website to your browser.

4. What is the difference between a regular tag and an empty tag?

Regular tags contain content, while empty tags do not need to contain content.
